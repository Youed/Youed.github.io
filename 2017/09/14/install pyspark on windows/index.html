在windows上面安装并用jupyter运行pyspark

Sep 14, 2017

由于个人的测试需要，打算在自己的pc上面安pyspark，来熟悉MLlib。
翻了一下bing和狗，方法不少。这里主要介绍下在windows下的安装（安装的时候总是出现各种问题，有条件的还是操作unix吧）
我是根据视频https://medium.com/@GalarnykMichael/install-spark-on-windows-pyspark-4498a5d8d66c来操作的，需要翻墙，有条件的同学可以看下，讲的特别详细。

Step1:安装Anaconda,GOW

前者的安装我就不写了。地址是
https://repo.continuum.io/archive/
我的机器上面装的miniconda。
GOW的地址：
https://github.com/bmatzelle/gow/releases/download/v0.8.0/Gow-0.8.0.exe
GOW的作用是在cmd下面执行linux 命令。安装结束之后可以运行：
gow –list
如果安装正确，可以返回

Step2：安装spark

spark的地址：
http://spark.apache.org/downloads.html
我选的是default的设置下的版本: spark-2.1.1-bin-hadoop2.7.tgz.
下载好后，记一下路径名称。

下面要确认java与hadoop已经安装（这里不再说了）
设置路径

这里一定要注意，spark与hadoop的版本要对应
到这一。步，pyspark就已经装好了
使用spark
pyspark –master local[2]
这个会打开ipython notebook

如果得到的输出是’1000000‘,说明系统正常。如果出现error，可以回头看一下路径是否正确。
